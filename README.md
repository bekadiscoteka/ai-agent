# ai-agent

A tiny, beautiful local AI agent — minimal code, maximal curiosity.

- Purpose: experiment with file-based agent workflows and helpers.
- No fluff. No cloud. Just code and coffee.

## Quick Start

With `uv` (recommended):

```bash
# install once: python -m pip install --user uv
uv install        # create venv and install deps (if configured)
uv run python main.py
```

Vanilla venv (fallback):

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt  # if you have one
python main.py
```
